Helpmate 0.1.2

https://github.com/rm-hull/helpmate.git
Released under the MIT License (MIT)

A Clojure library for templating/emitting SGML (HTML,SVG,XML) fragments.

Installation

To install, add the following dependency to your project or build file:

[rm-hull/helpmate "0.1.2"]
